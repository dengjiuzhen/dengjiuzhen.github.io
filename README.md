# dengjiuzhen.github.io

## Site Concept: A Gamified Personal Website  
A portfolio website that transforms browsing into an interactive, game-like experience—where visitors explore, collect, and engage with work across design and theory.

---

## Overview  
This project rethinks the traditional online portfolio format.  
Instead of presenting work through static pages or PDFs, this website creates a gamified environment where visitors move through different “spaces,” interact with content, and build their own collection of works.  

The project is inspired by:  
- museum navigation systems  
- collectible-based engagement (badges, souvenirs)  
- the idea that interaction can deepen attention  

While maintaining content clarity, the site introduces playful systems to make the viewing experience more memorable and personal.

---

## Experience Design  
The user experience follows a simple loop:  
**Explore → View → Collect → Earn → Unlock**

Visitors can:  
- navigate through different spaces (Studio, Archive, Café)  
- collect badges from works they find interesting  
- accumulate coins based on time spent on the site  
- spend coins in a café-like interaction space  
- unlock small rewards as a reflection of engagement  

The goal is not efficiency, but meaningful interaction and memory.

---

## Feature A: Badge Collection System  
*I want you to be able to find me again!*  

- Each project includes a “Collect Badge” interaction  
- Saved works appear in a personal Collection page  
- Enables quick revisiting of selected works  

---

## Feature B: Time-Based Currency System  
*I want to encourage you to engage~*  

- Visitors earn coins passively based on time spent on the site  
- Time persists across navigation within the site  
- Encourages exploration without pressure  

---

## Feature C: Café Interaction Space  
*I want to cheer you up :)*  

- Visitors can spend coins on virtual coffee  
- Each purchase generates a random cheerful message  
- Designed as a “pause” space within the experience  

---

## Feature D: Unlockable Reward  

After collecting a certain number of badges, users receive a small digital souvenir flower.  
This acts as a soft acknowledgment of engagement.

---

## Feature E: Collection Export  

- Users can export their collected works as a PDF  
- The PDF includes links for revisiting projects later  
- Functions as a personalized mini-portfolio  

---

## Site Structure  
/ → Lobby (entry point)
/studio → Design works (practice)
/archive → Theory works (papers)
/about → Bio + CV
/collection → Saved works (badges)
/cafe → Interactive rest space


---

## Visual Language  
The site uses a cyberpunk night aesthetic combined with soft, “affinity-driven” elements:

- Dark background for focus and immersion  
- Neon purple as primary identity color  
- Soft accent colors (pink, blue, green) for feedback and emotion  
- Rounded UI and glow effects to balance intensity with warmth  

---

## Technical Approach  
The system prioritizes lightweight interaction and accessibility over heavy frameworks:

- HTML / CSS / JavaScript  
- Local storage for:  
  - Badge collection  
  - Coin tracking  
- GitHub Pages for deployment
